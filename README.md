## Quick Start

```bash
# Install and build everything
npm run setup

# Start the application
npm start
```

**Access:** http://localhost:5038

- **Backend**: Node.js, Express, SQLite
- **Frontend**: React, Axios
- **Authentication**: JWT
- **Database**: SQLite (embedded)
